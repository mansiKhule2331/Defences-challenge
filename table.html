HTML Tables

What is a Table?

A table in HTML is used to display data in rows and columns. It is useful for organizing information such as student marks, product details, or employee records.

Important Tags:

  • <table> → Defines the table
  • <tr> → Defines a row
  • <td> → Defines a cell (table data)
  • <th> → Defines a header cell (bold by default)

Useful Attributes:

  • border → Adds border around table
  • cellpadding → Adds space inside each cell
  • cellspacing → Adds space between cells
  • colspan → Merges multiple columns
  • rowspan → Merges multiple rows
